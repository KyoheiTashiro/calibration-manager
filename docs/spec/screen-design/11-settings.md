# 11. 設定・バックアップ

[画面設計 共通仕様・全体構成](./README.md) の一部。ステータス色・モーダル共通挙動・確認ダイアログ等の共通仕様は [README(§0 共通仕様)](./README.md#0-共通仕様) を参照。

**目的**: PWAとしてのインストール、CSVによるデータのエクスポート/インポート、データ全削除。

**URL**: `/settings`

## 表示項目 / 機能

- **CSVエクスポート**: エンティティ(Equipment / InspectionItem / InspectionRecord / CalibrationOrder / Vendor / Person / Notification)ごとに個別ダウンロードボタン。文字コード **UTF-8 BOM付き**(Excel互換、ドメインモデル §5)。ファイル名例 `equipment_YYYY-MM-DD.csv`。
  - **CSV列仕様(D-028)**: 各エンティティの列 = `store/types.ts` のフィールド宣言順、ヘッダ行 = フィールド名(英語キー)そのまま。boolean は `true`/`false`、optional 未設定は空セル。エクスポート・インポートで同一仕様(ラウンドトリップ保証)。改行コードは CRLF、引用は RFC 4180 準拠(`"` 囲み + `""` エスケープ)。
- **CSVインポート**:
  1. 対象エンティティ選択 + ファイル選択。
  2. パース後 **zodバリデーション**(各エンティティのスキーマ + 参照整合・ユニーク制約)。
  3. **エラー行のプレビュー表示**(行番号 + エラー内容)。行番号はファイル上の行番号(ヘッダ=1行目、データ先頭=行2)。取り込み可能行数とエラー行数を提示。
  4. エラー1件以上で「確定」ボタンを非活性化する(D-030、スキップ取り込みは不実装)。エラー0件のときのみ「確定」→ **確認ダイアログ**(既存データが置き換えられる旨を明記)後に全件取り込み。
  - **取り込み意味論(D-029)**: 確定 = 選択エンティティの Record を取り込み行で**全置換**(upsert しない)。検証は (a) 行単位 zod、(b) ファイル内ユニーク(全種 id、equipment は managementNo も)、(c) 外向き参照の存在チェックは現在ストアの他エンティティと突合。
- **PWAインストール(D-037)**:
  - `beforeinstallprompt`(非標準・Chromium系のみ)をアプリ起動時に捕捉して `preventDefault()` し、[アプリとしてインストール]ボタン押下時に任意タイミングで `prompt()` を表示する(捕捉タイミングをアプリ起動時に統一する経緯は D-040)。
  - standalone 表示中(`display-mode: standalone`)、または `appinstalled` イベント発火後は「インストール済みです」を表示する。
  - イベント非対応ブラウザ(iOS Safari / Firefox 等)ではブラウザ標準機能からのインストール手順を案内文で表示する(Chrome/Edge: アドレスバーのインストールアイコン、iOS Safari: 共有メニューの「ホーム画面に追加」)。
- **データ全削除(D-031)**: **2段階確認**([§0.6](./README.md#06-確認ダイアログポリシー))。「全データを削除することを理解しました」チェック → 「削除」ボタン活性化 → 実行で LocalStorage を初期化。追加の多重確認(ConfirmModal 等)はしない。

## 表示ルール・バリデーション(zod)

- インポート各行を対応エンティティの zod スキーマで検証。必須列欠落・enum不正・日付形式(`YYYY-MM-DD`)不正・数値変換不可・ユニーク重複(managementNo 等)・参照先不存在(manufacturerId/personId/vendor/inspectionItemId 等)をエラーとして行単位で報告。
- エラー行が1件でもあれば「確定」は非活性(D-030、中断のみ・スキップ取り込みは不実装)。1件もエラーがなければ確認後そのまま取り込み。

## 空状態

- データ未登録時もエクスポートボタンは表示(ヘッダのみのCSVを出力)。
- インポート未選択時: プレビュー領域に「CSVファイルを選択すると、ここに検証結果が表示されます」。ファイル選択はネイティブ input を隠し、共通 Button(secondary)「ファイルを選択」+ 選択中ファイル名(未選択時「未選択」)で表示する。
