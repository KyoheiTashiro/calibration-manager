# 2. 機器一覧

[画面設計 共通仕様・全体構成](./README.md) の一部。ステータス色・モーダル共通挙動・確認ダイアログ等の共通仕様は [README(§0 共通仕様)](./README.md#0-共通仕様) を参照。

**目的**: 登録機器を一覧・検索し、詳細への入口とする。

**URL**: `/equipment`

## 画面レイアウト

## 表示項目

Equipment を主体とし各行に: `管理番号`(managementNo)、`機器名`(name)、`型式`(model)、`メーカー名`(manufacturerId → Vendor.name)、`設置場所`(location)、`状態`(status: active=稼働 / suspended=休止 / retired=廃棄)、`項目数`(当該機器の ServiceItem 件数)、`最も近い次回期限`(有効項目の nextDueDate 最小値。項目なし・非稼働は `—`)。

## 操作・アクション

- 検索ボックス: managementNo / name / model の部分一致(大文字小文字無視)。
- 状態フィルタ(5択、D-010): `稼働+休止`(既定)/ `全て` / `稼働` / `休止` / `廃棄`。フィルタ値は EquipmentStatus とは別軸の画面ローカル定数。
- 行クリック → `/equipment/:id`。
- 「+ 機器を追加」→ `/equipment/create`。

## 表示ルール・バリデーション

- 既定並び: managementNo 昇順。
- 最寄り期限のセルは、その機器の最小ステータス優先度に応じてバッジ色を付してよい(任意)。
- 入力バリデーションなし(検索・フィルタのみ)。

## 空状態

- 「機器が未登録です」+「+ 機器を追加」ボタン。
- 検索結果0件時: 「条件に一致する機器はありません」。
