# 3. 機器登録・編集

[画面設計 共通仕様・全体構成](./README.md) の一部。ステータス色・モーダル共通挙動・確認ダイアログ等の共通仕様は [README(§0 共通仕様)](./README.md#0-共通仕様) を参照。

**目的**: 機器(Equipment)の新規登録および編集。

**URL**: 登録 `/equipment/create` / 編集 `/equipment/:id/edit`(D-039)

## 画面レイアウト

```
┌──────────────────────────────────────────────┐
│ 機器を追加 / 機器を編集                       │
│ 管理番号*   [EQ-001            ]  ← ユニーク  │
│ 機器名*     [ノギス            ]              │
│ 型式        [CD-15             ]              │
│ シリアル番号 [1234567          ]              │
│ メーカー    [ミツトヨ        ▼] ← 製造元のみ │
│ 設置場所    [検査室            ]              │
│ 状態*       [稼働 ▼]                          │
│ 備考        [                  ]              │
│                                              │
│ [廃棄にする(編集時のみ)]    [キャンセル][保存]│
└──────────────────────────────────────────────┘
```

## 表示項目 / 入力フィールド

Equipment の属性に対応(ドメインモデル §3.3):

| フィールド   | 属性           | 必須 | 入力形式                                       |
| ------------ | -------------- | ---- | ---------------------------------------------- |
| 管理番号     | managementNo   | ○    | テキスト(ユニーク)                             |
| 機器名       | name           | ○    | テキスト                                       |
| 型式         | model          |      | テキスト                                       |
| シリアル番号 | serialNo       |      | テキスト                                       |
| メーカー     | manufacturerId |      | セレクト(Vendor で `isManufacturer=true` のみ) |
| 設置場所     | location       |      | テキスト                                       |
| 状態         | status         | ○    | セレクト(active/suspended/retired)             |
| 備考         | note           |      | テキストエリア                                 |

## 操作・アクション

- 「保存」→ 検証通過で作成/更新し `/equipment/:id`(詳細)へ遷移。
- 「キャンセル」→ 直前画面(一覧または詳細)へ戻る。
- 「廃棄にする」(編集時のみ表示。D-011): status が既に `retired` でも表示する(操作は冪等)。**確認ダイアログ**([§0.6](./README.md#06-確認ダイアログポリシー))後に `setEquipmentStatus(retired)` へ変更(物理削除しない。ドメインモデル §3.3)し、機器詳細(保存時と同じ遷移先)へ遷移する。この操作によるフォームの未保存編集は破棄する(保存とは独立した操作)。

## 表示ルール・バリデーション(zod)

- `managementNo`: 必須・非空。**ユニークチェック**(自身以外の Equipment と重複不可 → 「この管理番号は既に使用されています」)。
- `name`: 必須・非空。
- `status`: 必須・enum(`active`|`suspended`|`retired`)。既定 `active`。
- `manufacturerId`: 任意。指定時は存在する Vendor 参照であること。
- その他(model/serialNo/location/note): 任意・文字列。

## 空状態

- メーカーセレクトに `isManufacturer=true` の Vendor が0件: 「メーカーが未登録です。マスタから追加してください」+ `/vendors` リンク(空選択で保存可)。
