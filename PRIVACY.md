# Privacy Policy / プライバシーポリシー

最終更新日: 2026-07-22 / Last updated: 2026-07-22

本アプリ(calibration-manager)は、機器の点検・校正期限を管理するブラウザアプリです。

## データの収集 / Data Collection

本アプリは利用者の個人情報・利用状況を一切収集しません。

- 入力されたデータ(機器情報・点検記録など)はすべて利用者のブラウザ内(LocalStorage)にのみ保存されます
- サーバーへの送信は行いません(バックエンドが存在しません)
- アクセス解析・広告・トラッキング Cookie は使用していません

This app does not collect any personal information or usage data. All data
you enter is stored only in your browser's LocalStorage. There is no backend
server, and no analytics, ads, or tracking cookies are used.

## ホスティング / Hosting

本アプリは GitHub Pages で配信されています。配信に伴い GitHub がアクセスログ
(IP アドレスなど)を取得する場合があります。詳細は
[GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)
を参照してください。

This app is hosted on GitHub Pages. GitHub may collect access logs (such as
IP addresses) as part of serving the site. See the
[GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)
for details.

## オフラインキャッシュ / Offline Cache

本アプリは PWA としてオフライン動作するため、アプリ本体を利用者の端末に
キャッシュします。キャッシュは端末内にのみ保存されます。

As a PWA, this app caches its assets on your device for offline use. The
cache is stored only on your device.

## データの削除 / Data Deletion

保存されたデータはブラウザのサイトデータ削除(LocalStorage のクリア)で
すべて削除できます。開発者側に削除を依頼する必要はありません(開発者は
データを保持していません)。

You can delete all stored data by clearing the site data (LocalStorage) in
your browser. There is nothing to request from the developer, as the
developer holds no data.

## 変更 / Changes

本ポリシーの変更は本ファイルの更新をもって行います。変更履歴はリポジトリの
コミット履歴を参照してください。

Changes to this policy are made by updating this file. See the repository's
commit history for the change log.
